The "introNetworkingProgramming" folder contains example python networking code
to better understand the sections of the project.

The examples are in client/server pairs:
chatClient         <------> chatServer
pythonSocketClient <------> pythonSocketServer
srtclnt            <------> strsvr
wps                <------> svr
tmc                <------> tms

-------------------------------------------------------------------------------
Client: chatClient.py
Server: chatServer.py
Website Example: https://www.binarytides.com/code-chat-application-server-client-sockets-python/

-------------------------------------------------------------------------------
Client: pythonSocketClient.py - Communicates with www.google.com
Server: pyhtonSocketServer.py - Use with Telnet localhost 8888
Website Example: https://www.binarytides.com/python-socket-programming-tutorial/
